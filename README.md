# RPG Turns

System-agnostic Tabletop RPG combat turn tracker

![screenshot](assets/screenshot.png)

## Installing

```
npm install
or
yarn
```

## Running

```
npm start
or
yarn start
```

The app will be available on http://localhost:3000

## Testing

Run cypress
```
npm run cypress open
or
yarn cypress open
```
Click on `run all specs` button.
